Ativar virtual env:
cd projeto
cd venv
cd Scripts
activate

Criar projeto django
django-admin startproject nome do projeto

Rodando o codigo
cd caminho do projeto
python manage.py runserver

Tabelas do banco de dados sejam criadas
cd pasta do projeto
python manage.py migrate

Criar super usuario
python manage.py createsuperuser

Parar o servidor
ctrl + c
